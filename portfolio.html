Multi Product Fuel Dispenser

At Stuffbits we have developed firmware for all modules of a Multi-Product Fuel Dispenser. The product has a main CPU card which communicates to several slave cards on a CAN bus.

Slaves include Static Displays which retain value even after loss of power, LCD + Keypad for input and configuration, Non resetable Totalizer for Fuel Dispensed till date, Accurate and fast Encoder to measure actual fuel dispensed.

The product aslo has support for Remote Fueling and logs can be taken on MMC card. A printer to print receipts and a BMS.

This product meets the very stringent standards and requirements for Fuel Dispenser. The latest model includes 256-bit AES encryption on PKI infrastructure and is certified by CDAC.

Windmill Remote Monitoring

This Device monitors parameters of a windmill and sends data over GSM, to PC as required . Parameters are displayed over 3 pages, 3 pages for configuration setup, 3 blind totalized values. Peak/Totalizer values can only be reset with command by admin mobile or by special keypress sequence

This system has a Quectel M66 Modem on board. The code handles several connection related issues such as loss of network, 2G to 4G transition, Network not found.

Admin number and User number is settable via single key. On receiving a prompt from Admin or user number on SMS, device can reply with the necessary details.

Fleet Tracking Solution

A Fleet Tracking solution consists of a hardware and software that helps keeping track of the vehicles in fleet. Using the power of Google maps with continous GPS updates over 2G/4G network this solution is a reliable and effective method for keeping track of vehicles.

The firmware for the same was developed by us which included Ublocks GSM/GPRS module, LIS3DH accelerometer for motion sensing, NEO-M8N for GPS using STM32 Low power controller.

The device sends continous co-ordinates to the Server over MQTT when the vehicle is in motion. While resting the device goes in Low power mode to conserve battery.

This device also boasts of features such as file transfer using FTP, Bootloader with provision to upgrade firmware Over-The-Air.

Soil Sensor Prototype

A simple soil parameters sensor display module was developed to evaluate the JXCT soil sensors over Modbus

Hardware has STM32 microcontroller, 2.8" TFT display and 4 keys. RS485 for communciation.

This evaluation board supports 4 different soil sensors as follows

  • JXBS-3001-NPK-RS Soil NPK
  • JXBS-3001-PH-RS Soil
  • JXBS-3001-TR-RS
  • JXBS-3001-TH--RS

This evaluation board can display readings from upto four different soil sensors simulataneously on differenet pages.

The Modbus Address for each device is settable and model can be selected from the above list..

Air Purifier Controller

Air purifiers essentially work by sanitizing the air by using electric air filters having high-voltage electrodes.

A very compact and low cost controller was developed for this purpose. In built SMPS so that device operates directly on 230VAC.

It has Relay output to control high-voltage electrodes. While a Triac based speed control is applied to 2 AC fans inside the unit.

It has 4 selectable modes on rotary switch and 3 selectable fan speed on rotary switch

Parking Sensor

The Parking Sensor overhead indicator system is a simple, cost-efficient and highly effective off-street parking management system. This system transforms the multi-storey parking experience for drivers by providing high-visibility, colour coded LED overhead guidance indicators that are dynamically controlled by your smart parking business rules, allowing users to see the availability of parking space.

The parking sensor is based on Ultra-Sonic sensor for Obstacle detection and use High wattage RGB Led for indication.

Upto 128 devices can be connected on Modbus at a time to get centralized reading of Parking space availability.

A PC configuration tool on Modbus was also developed to set initial thresholds and device ID, color code for these sensors.
